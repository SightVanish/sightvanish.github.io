FineBI Quick Start

What is FinBI

FineBI是商业自主分析软件，类似大众熟悉的Tableau和PowerBI，相比于使用Excel的优势在于

  1. 轻量，速度快（部署在远端服务器，本地的操作全部在网页完成）
  2. 支持数据库同步（链接数据库，数据下载和Query查询都可以实现，而且更快更简单）
  3. 丰富的对图/表的支持，和Excel透视表功能类似，同时已经构建了大量常用功能

今天和大家做一个使用分享，简要介绍接FineBI的功能和优势特点。

LOGIN

左侧栏中用户常用功能是My Analysis和Public Data部分。My Analysis中保存了所有的看板和图表，主要的分析功能都会集中在这里。Public Data中是数据源，所有My Analysis中的看板数据都出自Public Data。

Public Data

数据分析所有的数据源全部来源于Public Data

  1. 找一个文件夹/新建一个文件夹，新建Database，SQL，Excel。Database为链接数据库里的某张表，SQL支持写代码（目前需要使用postgre sql语法），Excel支持手工上传线下Excel表。
    1. 选择Database可以选中自己需要的表；如果需要的数据不存在，可以联系IT，我们可以根据需求构建一张宽表，包含需要使用到的所有字段。
    2. 如果想要自己写Query查询，可以选择SQL，右上角可以点击Preview预览查询的数据。
    3. 对于线下手工维护的数据也可以通过Excel的方式上传到数据库中保存。
  2. 需要特别注意的是：
    1. FineBI的数据不会和数据库同步更新，需要手动设置更新方式。点击Update Information，选择数据更新方式。数据中台一般会在每日凌晨抽取数据，FineBI中可以考虑配置在每日8点左右更新数据。
  3. 打开My Analysis，点击New subject建立数据看板，在public data中选择对应的数据源即可。

Public Data首先可以满足大家数据下载的需求，用Excel透视表分析首先就需要跑一段Query把数据查出来；FineBI可以直接连接到数据库，临时想查数/计算指标可以直接在线上完成，需要深入的分析也可以下载所有的数据明细到本地excel进一步加工。

My Analysis

Analysis就是Excel中使用的数据透视表。它分为3个部分：Data，Component，DashBoard。

  1. Data负责数据，点击Data旁的“+”号增加新的表。 对于同一张表可以进行新增/减少字段，grouping，filter，多张表join和union等一系列操作。最右侧的竖栏会记录每一次操作，因此基本没有误操作风险，可以随意回退。

  2. 之后进入到Component。

    Component是Analysis的核心，本质同Excel透视表逻辑相同。通过拖拽左侧的字段到Dimension，Indicators中自动完成数据透视功能。

    需要着重提到的特点有两个：

    1. Component支持了多种图表类型，包括交叉表，明细表，折线图，饼图，玫瑰图等等
    2. 常用的指标计算方式都已经内置，比如计算百分比，均值，同比环比等
    3. 没有支持的计算方式可以通过Calculation Indicators完成，比如计算加权平均数。
    4. 此外也可以自由修改字段名，排序，筛选条件，颜色等细节

  3. 报表分析最终产出的结果是Dashboard，Dashboard是通过不同的Component组合而成。

    1. 拖动图表到Dashboard上
    2. 添加Filters，需要手动指定Filter控制哪些表
    3. 也可以增加额外在component中增加文字，Tab，图片等

Collaboration

  1. Public Link，可以让其他人也能看得到，没有权限管理。
  2. 在My Analysis中点击Collaboration，赋予其他人可以浏览或修改

More

FineBI相比于Excel最大的优势就在于链接数据库，当数据量特别大几百万数据的时候，使用Excel分析，无论是下载数据，数据透视表都会很慢，但是FineBI可以提供一个快速查询的入口，在FineBI上链接数据库，做一些轻度的汇总筛选再下载到本地深入分析。
